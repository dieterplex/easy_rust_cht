結束了嗎？

簡單英語學 Rust 就這樣結束了。但是我還在這裡，如果你有什麼問題可以告訴我。歡迎在 Twitter 上聯絡我或者新增 pull request、issue 等。如果有些地方不容易理解，你也可以告訴我。簡單英語學 Rust 需要非常容易理解，所以請告訴我英語太難的地方。當然 Rust 本身也可能是很難理解的，但我們至少可以確保英語是容易的。
